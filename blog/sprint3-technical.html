An analogy to describe the differences between HTML and CSS

Html is like your main content and structure when writing a report, and css is the font styling, colors and spacing that are added to wow and audience.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones).

Control flow in code is the order in which commands are run, with javascript code is run top to bottom, and commands run in order. It's like putting your milk in after your cereal because if it was the other way around it would be chaos.

Loops are used so that we can run code multiple times depending on conditions this is like washing dishes, as long as we have dishes to wash we should continue the process of running water, adding detergent and scrubbing.

Describe what the DOM is and an example of how you might interact with it.

The DOM Stands for Document Object Model and is essentially a family tree of all the elements that are contained on a webpage. We can navigate the tree by specifying elements by multiple factors and then interacting with them.

Explain the difference between accessing data from arrays and objects.

Arrays have a simple index that can be used to navigate them and find content while objects have a key, value pair where the key can be any type of variable. objects can be navigated using dot notation or square bracket notation similar to arrays.

Explain what functions are and why they are useful.

Functions are away of breaking up code into blocks so that code can be run multiple times without having to rewrite the same thing over and over.
