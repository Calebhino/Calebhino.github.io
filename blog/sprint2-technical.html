Margins, Borders and Padding

Every element on an HTML page is essentially contained inside a box/rectangle. Every box has the 3 properties: Margin, Border and Padding - which allow you to frame the box using spacing and visual border elements.

Margin

With the box model there are 3 layers to every box. The Outer layer is the Margin it's like the wrapping paper surrounding a present at Christmas.
The Margin allows you to add spacing between the outside of the border (the box our gift is in) and any other boxes on our HTML page.
Margin can be individually controlled for each side of the box so you can perfectly control the spacing across the page.

Border

If the Margin is our wrapping paper then the border is the box our gift is in. The Border is pretty simple, it's normally a visual element that can be different colours or different thicknesses and can even be dotted or have other effects. It's like a frame of a painting, except we're sticking to one analogy at this point.

Padding

The Padding is like the packing peanuts or styrophome inside the box, it acts as another way of adding spacing for our elements, but instead it adds spacing between the border (packaging box) and the elements inside our box i.e the magical present you bought me.
Padding can be individually controlled for each side much like margin.
